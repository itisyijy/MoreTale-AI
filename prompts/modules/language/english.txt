English language module.

이 모듈은 영어가 동화의 주 또는 부 언어일 때 영어 텍스트 작성 디테일을 제공한다.

## Apply when
- primary_lang 또는 secondary_lang에 English가 포함될 때 주입.
- 한국 다문화 가정에서는 영어가 secondary로 사용되는 경우가 많음(부모 한쪽이 미국·캐나다·호주·필리핀 출신 등). primary로 영어가 설정되는 경우는 해외 거주·이중언어 환경.

## Educational goals
- 초등 영어과 핵심역량(영어 의사소통, 자기관리, 공동체) 지원.
- 다문화 가정 자녀의 영어 노출 — 부모 한쪽의 모어이거나 가족 공용어인 경우 가정 언어로서의 따뜻함 모델링.
- 영어 학습 동기 부여 — "시험 영어"가 아닌 "이야기 속 살아 있는 영어"로 친숙도 형성.
- 다문화 정체성과 영어 사용을 연결 — 영어 모어 부모를 둔 자녀가 부모의 언어와 자신의 정체성을 자연스럽게 통합.

## Substantive guidance

When English is the primary language:
- 자연스러운 picture-book English. 라틴계 어휘 대신 앵글로색슨 어휘 우선("began" not "commenced", "asked" not "inquired").
- 문장 길이 권장:
  - 만 3–5세: 5–8 단어
  - 만 6–8세: 8–14 단어
  - 만 9–12세: 10–18 단어
- 읽어주기 리듬: short-short-long 패턴으로 변주. 문장 종결 부호는 리듬을 받쳐주도록 — em dash(—)는 호흡, ellipsis(…)는 잔류, exclamation(!)은 최소 사용.
- 의성어는 대문자로 강조: WHOOSH, THUMP, SNAP, CRACK, POP, SWISH, RUSTLE, BOOM, KNOCK KNOCK.
- 학습자에게 부담을 주는 phrasal verbs("put up with", "look after") 회피. 단일 동사 대체 사용.

When English is the secondary language (자녀가 학습 중):
- 시제는 단순 현재 또는 단순 과거 위주. 완료형(have/had been doing) 회피.
- 축약형 사용은 학습 단계에 따라 조절: 초기 학습자에게는 "do not"이 명확. 중급 이상에는 "don't"이 자연스러움.
- 문화 특화 관용구("piece of cake", "raining cats and dogs")는 같은 문장 내 풀이 없으면 사용 금지.
- 추상적 감정어보다 구체적 감각 동사 우선 사용.
- 어휘 헤드워드는 Tier-1 또는 초기 Tier-2 빈도. 정의는 단순 SVO.

Korean mimetic / onomatopoeia handling (한국어가 짝 언어일 때):
한국어는 의성어(sound-symbolic)와 의태어(manner-symbolic)가 매우 풍부하지만 영어는 의성어 중심이고 의태어가 거의 없다. 한쪽 언어를 다른 쪽으로 직역하면 품사 충돌·의미 왜곡이 자주 발생한다. 다음 규칙을 적용한다:

1. 분류 먼저, 변환 다음. 한국어 표현이 (a) 의성어인지(소리 모방), (b) 의태어인지(모양·상태 모방), (c) 부사적 강조인지 먼저 판별한다.

2. 의성어 → 영어 의성어로 대응. 같은 소리를 모방하는 영어 표현이 있을 때만 대문자로 표기.
   - 똑똑 → KNOCK KNOCK ✓
   - 짝짝 → CLAP CLAP ✓
   - 두근두근 → THUMP-THUMP ✓ (심장 박동 소리)
   - 슈웅 → SWOOSH ✓

3. 의태어 → 영어 부사/형용사/동사구로 풀어쓰기. 의태어를 대문자 의성어로 직역 금지.
   - 꼼꼼(하게) → "carefully" / "meticulously" (소리 아님; CRUMPLE 같은 의성어 금지)
   - 휘둥그레 → "eyes opened wide" / "in wonder" (소리 아님; WOW 같은 감탄사 금지)
   - 활짝(피다) → "bloomed wide open" / "burst into a smile" (소리 아님)
   - 물끄러미 → "stared quietly" / "gazed in silence"
   - 알록달록 → "colorful" / "in a rainbow of colors"
   - 풀이 죽은 → "downcast" / "with slumped shoulders"

4. 모호한 경우(의성·의태 양쪽 가능): 한국어 문맥의 우세 감각을 따른다. 예: "쓱싹쓱싹"이 연필 소리이면 SCRITCH SCRATCH(의성), 행동의 리듬 묘사면 "back and forth"(의태).

5. 한 페이지에서 한국어 의성·의태어를 사용했더라도 영어 측이 자연스럽지 않으면 강제로 의성어를 만들지 말고 산문으로 푼다. 부자연스러운 영어 의성어는 어린이 읽기 흐름을 깨뜨린다.

6. 영어가 primary일 때 한국어 측도 같은 원칙을 거꾸로 적용: 영어 의성어를 한국어 의태어로 임의 치환 금지. 한국어 측에서는 한국어 의성·의태어로 자연 표현을 우선.

## Multicultural context note
한국 다문화 가정에서 영어 모어 부모는 결혼이민자 통계에서 큰 비중은 아니지만(필리핀·미국·캐나다 등 합산 약 8%), 자녀의 영어 노출 환경이 가정과 학교 모두에서 비중이 큼.

필리핀계 영어는 영어와 타갈로그·세부아노 등이 혼재된 가정 환경을 가정. 부모가 영어 사용 시 형식적이지 않은 자연스러운 가정 영어 모델링.

미국·캐나다·호주계 영어는 지역색을 피하고 중립적 영어 사용 권장.

영어 학습이 자녀의 "정체성 표현 도구"가 되도록 — 시험 영어와 분리된 따뜻함을 유지.

## Avoid
- 영어 학습 교재 같은 단조로운 문체. 자연스러운 이야기 호흡이 목표.
- 영어 모어 부모를 "글로벌하다, 멋지다"는 우월적 프레이밍 — 한국어 모어 부모와 위계 형성 금지.
- 영어 능력으로 캐릭터의 지적 수준이나 매력을 차등화하는 묘사.
- 일부 발음 표기로 부모를 희화화하기(특히 비원어민 영어 발화의 풍자적 표기).
